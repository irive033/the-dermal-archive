# The Dermal Archive

A curated catalog of acne-safe, non-comedogenic products — skincare, makeup, body,
hair, and tools, vetted against pore-clogging ingredients.

Static site: plain HTML + inlined CSS, no build step. Open `index.html` in a browser,
or serve the folder with any static file server.

## Structure

- `index.html` — the product list (shop) with category / subcategory filtering
- `routines.html` — Routines (blog homepage)
- `about.html` — about the archive + how products are vetted
- `contact.html` — contact form
- `products/` — individual product hero pages (one per item)

Product images and "Shop Now" links point to external retailers; Routines thumbnails
and posts currently load from the original site.
